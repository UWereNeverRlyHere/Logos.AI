# Logos.AI - Інтелектуальна База Знань (RAG)

**Logos.AI** — це система **Retrieval-Augmented Generation (RAG)**, побудована на **.NET 9**. Проект дозволяє завантажувати PDF-документи, автоматично розбивати їх на фрагменти (чанкінг), векторизувати текст та виконувати семантичний пошук для знаходження релевантної інформації.

Система використовує гібридний підхід до зберігання даних: реляційна база для надійності та векторна база для швидкого пошуку за змістом.

## 🚀 Основні можливості

*   **📄 Обробка PDF:** Завантаження документів через веб-інтерфейс, екстракція тексту зі збереженням номерів сторінок.
*   **🧩 Розумний Чанкінг (Chunking):** Розбиття тексту на логічні блоки з перекриттям (overlap) для збереження контексту.
*   **🧠 Семантичний Пошук:** Використання OpenAI Embeddings для пошуку за змістом, а не за ключовими словами.
*   **⚡ Векторна БД:** Інтеграція з **Qdrant** для високошвидкісного пошуку схожих векторів.
*   **💾 Гібридне сховище:**
    *   **SQLite (EF Core):** "Холодне" сховище для оригінальних текстів, метаданих та історії завантажень.
    *   **Qdrant:** "Гаряче" сховище векторів та payload для пошуку.
*   **🤖 Reasoning:** Підтримка аналізу контексту за допомогою LLM (`gpt-4o-mini`).
*   **🌐 API & UI:** ASP.NET Core MVC інтерфейс та REST API контролери з підтримкою `Newtonsoft.Json`.

### Backend
* **Core:** .NET 9 (ASP.NET Core MVC)
* **Architecture:** Clean Architecture (API, Engine, Abstractions)
* **Database:** PostgreSQL / MSSQL (через EF Core) для метаданих.

### AI & Data
* **Embeddings:** OpenAI `text-embedding-3-small` (1536 dimensions).
* **Vector DB:** Qdrant (зберігання векторів та payload).
* **PDF Processing:** PdfPig (для витягування тексту та координат).

## ⚙️ Налаштування (appsettings.json)

Система використовує гнучкий паттерн Options для налаштування параметрів RAG без перекомпіляції:

```json
{
  "Rag": {
    "ChunkSizeWords": 300,    // Цільовий розмір блоку тексту
    "ChunkOverlapWords": 50,  // Перекриття для збереження контексту
    "MinScore": 0.5,          // Поріг схожості (фільтр шуму)
    "TopK": 5                 // Кількість результатів
  },
  "Qdrant": {
    "Host": "localhost",
    "Port": 6334,
    "CollectionName": "logos_knowledge_base"
  },
  "OpenAI": {
    "ApiKey": "sk-...",
    "EmbeddingModel": "text-embedding-3-small"
  }
}
