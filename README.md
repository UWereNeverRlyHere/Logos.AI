# Logos.AI

Централізована бібліотека та сервіси для проєкту Logos.AI.

## Опис проєкту
Даний репозиторій містить основні компоненти системи Logos.AI, включаючи абстракції, ключову логіку та веб-інтерфейси. Проєкт орієнтований на використання сучасних технологій .NET.

## Структура рішення
*   **Logos.AI.Core** — Основна логіка та веб-додаток (Blazor/ASP.NET Core).
    *   `Logos.AI.Core.Web` — Фронтенд та API.
    *   `Logos.AI.Core.AppHost` — Хост для .NET Aspire.
    *   `Logos.AI.Core.ServiceDefaults` — Загальні налаштування сервісів.
*   **Logos.AI.Abstractions** — Загальні інтерфейси та контракти.

## Технологічний стек
*   **Платформа:** .NET 10.0
*   **Мова:** C# 14.0
*   **UI:** Blazor
*   **Оркестрація:** .NET Aspire

## Як почати роботу
1.  Переконайтеся, що у вас встановлено **.NET 10 SDK**.
2.  Відкрийте файл `Logos.sln` у **JetBrains Rider** або **Visual Studio**.
3.  Запустіть проєкт `Logos.AI.Core.AppHost` для локальної розробки.

## Конфігурація
Загальні налаштування збірки (версія мови, TargetFramework, Nullable-перевірки) керуються централізовано через файл `Directory.Build.props`.
